📖 Usage Guide
For Cashiers (Main POS)
Open index.html in your browser
Search Products - Type in the search box or use category filters
Add to Cart - Click products or use Quick Add buttons (Alt+1-5)
Scan Barcode - Click "📱 Scan Barcode" button (F2) to scan product codes
Adjust Quantities - Use +/- buttons in the cart
Checkout - Click "Checkout" to complete the sale
Print Receipt - Print the receipt from the modal window
For Administrators
Open admin-login.html and enter credentials
Access the dashboard to:
View key business metrics
